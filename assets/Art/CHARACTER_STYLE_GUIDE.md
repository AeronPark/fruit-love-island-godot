- 3D rendered, Pixar/Disney-esque

- Swimwear/summer beach attire

## Generation Rules

### For new expressions (image-to-image):
1. Use the character's reference image as `--image`
2. Only describe the CHANGE, not the character
3. Keep solid dark-green background for easy removal

**Good prompt**: "sad expression, teary eyes, looking down"
**Bad prompt**: "strawberry head character, female, white bikini, sad expression" (re-describes = inconsistent)

### For new poses:
1. Same rules - reference image + minimal prompt
2. Describe pose/action only: "arms crossed, looking away, annoyed"

## Background Colors for Removal
- Use: solid dark-green, solid blue, solid gray
- Avoid: complex backgrounds, gradients
